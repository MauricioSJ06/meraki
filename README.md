# Meraki Dental Care Coyoacán

**Landing page boutique para consultorio dental en Coyoacán, CDMX.**

---

## 🌸 Descripción del Proyecto

**Meraki Dental Care Coyoacán** es una aplicación web de alta gama diseñada bajo el sistema de diseño **Serene Boutique Dental**. Combina el minimalismo editorial con una calidez táctil ("hygge") para transformar la percepción del cuidado dental tradicional en un oasis de tranquilidad, bienestar y precisión médica de vanguardia.

El sitio ofrece una experiencia fluida, completamente responsive y accesible, integrando comunicación instantánea vía WhatsApp, ubicación interactiva con Google Maps y una arquitectura limpia pensada para un rendimiento y SEO óptimos.

---

## 🛠️ Tecnologías Utilizadas

- **HTML5 Semántico**: Estructura limpia, accesible y optimizada para motores de búsqueda (SEO & Open Graph).
- **Tailwind CSS (vía CDN con configuración Custom)**: Implementación fiel del Design System con tokens de color, tipografía y formas a medida.
- **Vanilla JavaScript (ES6+)**: Lógica modular para barra de navegación adhesiva (`sticky + glassmorphism`), menú lateral móvil, animaciones de aparición al hacer scroll (`IntersectionObserver`), carrusel de testimonios responsive y acordeón interactivo para preguntas frecuentes.
- **Tipografía**:
  - *Bodoni Moda*: Títulos y encabezados editorial luxury.
  - *Manrope*: Cuerpo de texto claro y profesional.
- **Iconografía**: Material Symbols Outlined.

## 🚀 Cómo Desplegar

Al encontrarse el archivo principal `index.html` en la raíz del proyecto, el sitio está listo para desplegarse en segundos en cualquier plataforma de hosting estático:

### GitHub Pages (Recomendado)
1. Sube el código al repositorio de GitHub: `https://github.com/MauricioSJ06/meraki.git` en la rama `main`.
2. Ve a la pestaña **Settings** (Configuración) del repositorio.
3. En el menú lateral, selecciona **Pages**.
4. En **Build and deployment > Branch**, selecciona `main` y la carpeta `/ (root)`.
5. Haz clic en **Save**. En pocos minutos tu sitio estará en vivo.

### Netlify o Vercel
1. Conecta tu repositorio de GitHub o arrastra la carpeta del proyecto en el panel de despliegue.
2. Configura el directorio de publicación en la raíz (`./`).
3. No se requiere comando de compilación (Build command en blanco).
